📋 PASSO A PASSO SIMPLES:

1. ABRA O DOCKER DESKTOP
   - Procure "Docker Desktop" no menu iniciar
   - Clique para abrir
   - Aguarde aparecer: "Docker Desktop is running"
   ⏱️  Isso pode levar 1-2 minutos

2. EXECUTE O ARQUIVO start.bat
   - Dê dois cliques no arquivo "start.bat"
   - Ele fará tudo automaticamente:
     ✓ Subir banco de dados
     ✓ Criar tabelas
     ✓ Popular com dados
     ✓ Iniciar servidor

3. ABRA NO NAVEGADOR
   - Acesse: http://localhost:3000
   - Email: admin@demo.com
   - Senha: admin123

❓ TEM PROBLEMAS?

Se o Docker Desktop não abre:
→ Leia: COMO-RODAR-LOCAL.md (Opção 2)
→ Instale PostgreSQL manualmente

Se aparecer erro "port 5432 already in use":
→ Você já tem PostgreSQL rodando
→ Veja COMO-RODAR-LOCAL.md seção "Problemas Comuns"

Se aparecer erro "Prisma Client":
→ Abra terminal aqui e execute: npx prisma generate
